# Portfolio - Abdoulaye Mahamat

## Description
Portfolio professionnel d'Abdoulaye Mahamat, étudiant en génie logiciel à Douala. Ce site web présente mes compétences, projets et informations de contact dans le domaine du développement web.

## Structure du projet
Le projet est organisé en 5 fichiers principaux :

1. **index.html** - Page principale du portfolio
2. **style.css** - Feuille de style principale avec design responsive
3. **script.js** - Fonctionnalités JavaScript interactives
4. **projects.js** - Gestion dynamique des projets
5. **README.md** - Documentation du projet

## Images utilisées
- **MMMM.jpg** - Photo de profil d'Abdoulaye Mahamat
- **BBB.jpg** - Projet 1: Site Web Restaurant
- **DDD.jpg** - Projet 2: Système de Gestion de Stock
- **PPP.jpg** - Projet 3: Portfolio Professionnel

## Instructions d'utilisation
1. Téléchargez les 5 fichiers (index.html, style.css, script.js, projects.js, README.md)
2. Placez vos images (MMMM.jpg, BBB.jpg, DDD.jpg, PPP.jpg) dans le même dossier
3. Ouvrez le fichier `index.html` dans un navigateur web
